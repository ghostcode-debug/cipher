# 🔐 Cipher - Privacy-Focused Messaging Platform

A cross-platform messaging application with privacy and security as core features.

## Features

- **End-to-End Encryption** - Optional message encryption
- **Privacy Controls** - 10+ privacy settings
- **Multi-Platform** - Web, Desktop (Windows/Mac/Linux), Mobile (iOS/Android)
- **Open Source** - MIT Licensed
