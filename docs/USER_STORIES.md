# User Stories

## User Personas

### Persona 1: Privacy-Conscious User (Alice)
- Goal: Communicate privately without being tracked
- Pain Point: Doesn't trust big tech companies
- Story: "As Alice, I want to send encrypted messages so that only my friend can read them"

### Persona 2: Open Source Developer (Bob)
- Goal: Build on top of open source tools
- Pain Point: Vendor lock-in
- Story: "As Bob, I want to self-host Cipher so I control all my data"

### Persona 3: Casual User (Carol)
- Goal: Easy-to-use messaging
- Pain Point: Complex privacy settings
- Story: "As Carol, I want simple one-click privacy so I don't have to think about it"

## User Stories by Persona

### Alice's Stories
1. Send encrypted message to friend
2. Verify message is encrypted (see badge)
3. Block unwanted contact
4. Clear message history
5. Set self-destruct timer

### Bob's Stories
1. Self-host the server
2. Deploy on own infrastructure
3. Control encryption keys
4. Backup user data
5. Run security audits

### Carol's Stories
1. Open app and start chatting
2. Toggle privacy with one click
3. See when friend is online
4. Share photos easily
5. Get notifications
