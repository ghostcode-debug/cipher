# Development Tools & Environment

## Required Tools

### Git
- **Version:** 2.30+
- **Download:** https://git-scm.com
- **Purpose:** Version control
- **Installation:** Run installer, accept defaults
- **Verify:** git --version
- **Status:** Already installed

### Node.js
- **Version:** 14.x or higher (18.x/20.x recommended)
- **Download:** https://nodejs.org
- **Purpose:** JavaScript runtime, npm package manager
- **Installation:** Run installer, accept defaults
- **Verify:** node --version && npm --version
- **Status:** Already installed

### npm
- **Version:** 6.0+
- **Comes with:** Node.js
- **Purpose:** Package manager
- **Verify:** npm --version
- **Status:** Already installed

### Text Editor
- **Recommended:** Visual Studio Code
- **Download:** https://code.visualstudio.com
- **Purpose:** Code editing, debugging
- **Installation:** Run installer, accept defaults
- **Verify:** Open VS Code
- **Status:** Already installed

## Command Line Tools

### Bash/PowerShell
- **Windows:** PowerShell (built-in)
- **macOS/Linux:** Bash (built-in)
- **Purpose:** Terminal commands

### npm Scripts
- **Purpose:** Run custom commands
- **Location:** package.json scripts section

### Common Commands
- git init (Initialize git repo)
- git add . (Stage all files)
- git commit -m "msg" (Commit changes)
- git push (Push to remote)
- npm install (Install dependencies)
- npm start (Start development)
- npm run build (Build for production)
- npm test (Run tests)

## System Requirements

### Minimum (Web Only)
- OS: Windows 7+, macOS 10.13+, Ubuntu 14.04+
- RAM: 4 GB
- Disk: 5 GB free
- CPU: Any modern processor
- Internet: Required for npm packages

### Recommended (All Platforms)
- OS: Windows 10/11, macOS 11+, Ubuntu 20.04+
- RAM: 8 GB+
- Disk: 20 GB free

## Browser DevTools

### Chrome DevTools
- **Open:** F12 or Ctrl+Shift+I
- **Tabs:**
  - Elements (HTML/CSS inspection)
  - Console (JavaScript debugging)
  - Sources (code debugging)
  - Network (request inspection)
  - Performance (performance analysis)
  - Lighthouse (audits)

### Firefox DevTools
- **Open:** F12 or Ctrl+Shift+I
- Similar features to Chrome

### Safari DevTools
- **Open:** Cmd+Option+I (macOS)
- Enable in: Preferences > Advanced

## Keyboard Shortcuts (VS Code)

### Editing
- Ctrl+S: Save file
- Ctrl+Z: Undo
- Ctrl+Shift+Z: Redo
- Ctrl+X: Cut line
- Ctrl+C: Copy line
- Ctrl+V: Paste
- Ctrl+/: Toggle comment

### Navigation
- Ctrl+P: Open file
- Ctrl+Shift+P: Command palette
- Ctrl+G: Go to line
- Ctrl+H: Find and replace

### Terminal
- Ctrl+: Open terminal
- Ctrl+Shift+: New terminal

### Debugging
- F5: Start debugging
- F10: Step over
- F11: Step into
- Shift+F11: Step out
